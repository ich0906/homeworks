목차

  1. 1.메인 페이지 + 공통으로 사용된 기술  [Main 이동]
    • 1-1) html문서 구조화: section, article, nav 등 이용하여 작성
    • 1-2) 상단의 메뉴바: 리스트에 CSS 적용하여 작성(position, 크기, 배경, 여백, hover/click할 경우 글자색 변경 등)
      1-3) 또한 링크를 통해 메뉴 리스트의 요소들을 클릭하면 해당 내용이 작성된 다른 페이지로 이동하도록 하였습니다.(좌측 로고 클릭 시 메인페이지 이동 포함)
      1-4) 또한 각각의 메뉴마다 unordered list를 추가하여 평소엔 opacity=0, visibility=hidden으로 설정하여 보이지 않게 하다가 마우스를 올리면 opacity=1, visibility=visible로 바꾸게 하여 sub-menu를 구성하였습니다. sub-menu에 마우스를 올리면 글자색바뀜+밑줄 효과를 넣었습니다.
    • 1-5) 테이블: 테이블에 CSS를 적용하여 작성(position, 크기, 배경 등), 이미지 삽입하였습니다.
      1-6) "재생", "Info" 부분에 링크를 적용한 이미지 버튼 생성. 재생 버튼 클릭 시 해당 음악을 Youtube 링크를 통해 재생하도록 하였습니다.
      1-7) 테이블의 행에 마우스를 올리면 색깔이 바뀌도록 하였으며, 행을 클릭하면 해당 행의 색이 바뀌도록 onclick을 사용하여 구현하였습니다.
      또한 행을 더블클릭하면 ondblclick을 통해 info링크를 클릭한 것 처럼 새 창을 띄우도록 하였습니다.
    • 1-8) Info 링크: 자바스크립트의 window.open()함수를 이용하였으며 사진을 클릭 시 500x500 크기의 새 웹 페이지를 생성해 해당 곡의 정보를 보여주고, 댓글창을 구현하였습니다.
    • 1-9) 로그인 폼: 자바스크립트를 이용하였으며, 아이디와 비밀번호 입력 후 로그인 버튼을 클릭할 시 "아이디"님 환영합니다!를 출력하며 로그아웃 버튼이 생성되도록 하였습니다.

    • 해당 페이지에서 content에 마우스를 올릴 경우 사용한 기술이 보입니다.(title=""이용)

  2. 2."장르별" 페이지에 사용된 기술   [장르별 이동]
    • 2-1) 구조화, 메뉴바, 로그인 메인 페이지와 동일.
    • 2-2) 음악 테이블의 캡션 부분에 각 장르 페이지로 이동 가능한 버튼 생성.(현재 선택한 장르 버튼은 회색으로, 나머진 다른색으로 구현)
  3. 3."국내음악" 페이지에 사용된 기술   [국내음악 이동]
    • 3-1) 구조화, 메뉴바, 로그인 메인 페이지와 동일.
    • 3-2) 음악 테이블의 캡션 부분에 각 연령대 별 음악으로 이동 가능한 버튼 생성.(현재 선택한 버튼 배경색은 회색으로, 나머진 다른색으로 구현)
  4. 4."해외음악" 페이지에 사용된 기술   [해외음악 이동]
    • 4-1) 구조화, 메뉴바, 로그인 메인 페이지와 동일.
    • 4-2) 음악 테이블의 캡션 부분에 차트별 인기음악으로 이동 가능한 버튼 생성.(현재 선택한 차트 버튼 배경색은 회색으로, 나머진 다른색으로 구현)
    • 4-3) Bill Board 메뉴에서는 Bill Board 차트 사이트로, Spotify 메뉴에서는 Spotify 사이트로 이동하는 버튼 생성.(CSS 이용해 색상,테두리,패딩 등 설정)
  5. 5."아티스트" 페이지에 사용된 기술   [아티스트 이동]
    • 5-1) 구조화, 메뉴바, 로그인 메인 페이지와 동일.
    • 5-2) 음악 테이블의 캡션 부분에 국내/해외 아티스트 페이지로 이동 가능한 버튼 생성.(현재 선택한 차트 버튼 배경색은 회색으로, 나머진 다른색으로 구현)
    • 5-3) Info 버튼: 버튼을 누르면 600x600 사이즈의 새 페이지가 등장하고, 해당 artist의 이름이 페이지 title로 설정된다.
      -artist의 정보와 대표 앨범들이 이미지로 나열되어 있으며, 맨 아래에 속이 빈 하트그림을 클릭하면 색칠된 하트로 바뀌도록 구현하였다.(자바스크립트 이용)
    • 5-4) Link 버튼: 색칠되지 않은 하트 사진을 클릭하면 색칠된 하트 모양으로 바뀌도록 구현하여 Like표시를 할 수 있도록 하였다. 또한 색칠된 하트를 다시 클릭하면 원래의 색칠되지 않은 하트 사진으로 바뀌도록 하였다.(자바스크립트 이용.)
